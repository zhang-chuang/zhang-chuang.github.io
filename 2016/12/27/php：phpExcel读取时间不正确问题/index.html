php：phpExcel读取时间不正确问题

2016-12-27 | PHP   |   180 字 | | 阅读次数

用phpExcel读取上传的excel时，单元格内如果包含2015/05/05格式的时间读取的结果并不是想象 2015/05/05 而是变成了 45646这种

1：excelTime方法

/**
 * 用于处理excel中 2015/05/05读取不正确
 * @param $date excel中时间
 * @param bool $time 是否带时分秒(默认false)
 * @return array|int|string 2015-05-05
 */
public function excelTime($date, $time = false) {
    if(function_exists('GregorianToJD')){
        if (is_numeric( $date )) {
            $jd = GregorianToJD( 1, 1, 1970 );
            $gregorian = JDToGregorian( $jd + intval ( $date ) - 25569 );
            $date = explode( '/', $gregorian );
            $date_str = str_pad( $date [2], 4, '0', STR_PAD_LEFT )
                ."-". str_pad( $date [0], 2, '0', STR_PAD_LEFT )
                ."-". str_pad( $date [1], 2, '0', STR_PAD_LEFT )
                . ($time ? " 00:00:00" : '');
            return $date_str;
        }
    }else{
        $date=$date>25568?$date+1:25569;
        $ofs=(70 * 365 + 17+2) * 86400;
        $date = date("Y-m-d",($date * 86400) - $ofs).($time ? " 00:00:00" : '');
    }
    return $date;
}

使用

$this->excelTime('2015/05/05');
//输出结果 2015-05-05
