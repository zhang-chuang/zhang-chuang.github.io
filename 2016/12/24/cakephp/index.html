cakephp 设置与使用全局变量

2016-12-24 | PHP-cakephp   |   150 字 | | 阅读次数

在cakephp中有时候需要在所有的继承controller中公用这个变量，达到一处定义多处使用的目的；

在cakephp中有时候需要在所有的继承controller中公用这个变量，达到一处定义多处使用的目的；

在父级Controller内定义一个全局变量

class AppController extends Controller {

    public $sessionObj; //定义全局变量 本例这里用来存储session

    public function beforeFilter() {
        parent::beforeFilter();
        //给全局变量赋值
        $this->sessionObj = $this->Session->read('XXX');
    }

}

在继承类下使用全局变量

class BaseController extends AppController {
    public function index(){
        $this->sessionObj;
        //通过 $this->sessionObj; 拿到全局变量
    }
}
